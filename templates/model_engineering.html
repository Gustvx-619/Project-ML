03

Training Process

All 3 models are trained on the same 960-record training set. LR and KNN require Standard Scaling (mean=0, std=1) while Random Forest uses raw features.

from sklearn.preprocessing import StandardScaler from sklearn.ensemble import RandomForestClassifier from sklearn.linear_model import LogisticRegression from sklearn.neighbors import KNeighborsClassifier # Scale features (required for LR and KNN) scaler = StandardScaler() X_train_sc = scaler.fit_transform(X_train) X_test_sc = scaler.transform(X_test) # Model 1: Random Forest (uses raw features — scale-invariant) rf = RandomForestClassifier( n_estimators=100, max_depth=10, class_weight='balanced', random_state=42 ) rf.fit(X_train, y_train) # Model 2: Logistic Regression (uses scaled features) lr = LogisticRegression( C=1.0, max_iter=1000, class_weight='balanced', random_state=42 ) lr.fit(X_train_sc, y_train) # Model 3: KNN (uses scaled features) knn = KNeighborsClassifier(n_neighbors=7, metric='euclidean') knn.fit(X_train_sc, y_train) # Save all models for production use import joblib joblib.dump(rf, 'models/rf_model.pkl') joblib.dump(lr, 'models/lr_model.pkl') joblib.dump(knn, 'models/knn_model.pkl') joblib.dump(scaler, 'models/scaler.pkl')

08

Validation Process

5-fold cross-validation splits the training set into 5 equal parts, trains on 4 and tests on 1, rotating 5 times. The average score confirms the model generalizes well.

from sklearn.model_selection import cross_val_score # 5-fold cross-validation on the training set only rf_cv = cross_val_score(rf, X_train, y_train, cv=5, scoring='accuracy') lr_cv = cross_val_score(lr, X_train_sc, y_train, cv=5, scoring='accuracy') knn_cv = cross_val_score(knn, X_train_sc, y_train, cv=5, scoring='accuracy') print(f"RF CV scores: {rf_cv.round(3)} → mean: {rf_cv.mean():.4f}") print(f"LR CV scores: {lr_cv.round(3)} → mean: {lr_cv.mean():.4f}") print(f"KNN CV scores: {knn_cv.round(3)} → mean: {knn_cv.mean():.4f}")
RF CV scores: [0.893 0.896 0.891 0.901 0.901] → mean: 0.8958 LR CV scores: [0.828 0.823 0.818 0.833 0.833] → mean: 0.8271 KNN CV scores: [0.740 0.729 0.734 0.740 0.740] → mean: 0.7365

The small variance between CV folds (±0.5%) confirms the Random Forest model is stable and not overfitting to the training data.
