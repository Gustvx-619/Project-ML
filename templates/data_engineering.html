Phase 2 · CRISP-ML

Data Engineering

Cleaning, transformation, encoding, feature engineering, and normalization of the 1,200-record BIC dataset to maximize the model's predictive potential.

01

Data Cleaning

Strict logical rules were applied to ensure dataset integrity before modeling.

Rule 01
Correction of negative PM10 values
Negative values were detected in PM10_Promedio_ugm3 (min: -13.5 μg/m³), which are physically impossible. They are replaced with the valid minimum value of 0.
# Rule 1: PM10 cannot be negative df['PM10_Promedio_ugm3'] = df['PM10_Promedio_ugm3'].clip(lower=0) # Verification print(df['PM10_Promedio_ugm3'].min()) # → 0.0
Rule 02
Structural age range validation
All structures are validated to be between 50 and 450 years old — a historically plausible range for Colombian BICs. 3 extreme outliers were detected.
# Rule 2: Age between 50 and 450 years df = df[df['Edad_Estructura_Anios'].between(50, 450)] print(f"Valid records: {len(df)}") # → 1197
Rule 03
Null value verification
Confirmed that the dataset contains no NaN values in any column. Result: 0 nulls across all 11 variables.
# Rule 3: No null values print(df.isnull().sum()) # → all 0 assert df.isnull().sum().sum() == 0, "Error: null values found"
Rule 04
Relative humidity ≤ 100% validation
Verified that relative humidity does not exceed 100%, the absolute physical limit. The dataset complies (max: 99.4%).
# Rule 4: Humidity ≤ 100% assert df['Humedad_Prom_Anual_pct'].max() <= 100 print(f"Max humidity: {df['Humedad_Prom_Anual_pct'].max()}") # → 99.4

03

Feature Engineering

2 new variables are created to capture synergistic risks not represented by individual variables.

New feature 01
Acid Rain Index
Combines annual precipitation and PM10 to estimate the impact of acid rain on construction materials. The higher the precipitation and pollution, the greater the risk of chemical erosion.
# Feature 1: Acid Rain Index df['Acid_Rain_Index'] = ( df['Precipitacion_Anual_mm'] * df['PM10_Promedio_ugm3'] ) / 1000 # Mean: 40.61 | Std: 31.10 | Range: 0 – 153.07
New feature 02
Vibration Risk Index
Relates the age of the structure to its proximity to high-traffic roads. Older structures closer to roads have greater susceptibility to mechanical vibration damage.
# Feature 2: Vibration Risk Index df['Vibration_Risk'] = ( df['Edad_Estructura_Anios'] / df['Distancia_Via_Principal_m'] ).round(3) # Mean: 7.13 | Std: 16.79 | Range: 0.07 – 154.0

05

Normalization & Scaling

Numerical variables have very different ranges (2m to 672m vs. 51% to 99%). Min-Max Scaler is applied to bring all values to the [0, 1] range.

# Min-Max Normalization — numerical variables from sklearn.preprocessing import MinMaxScaler numeric_cols = [ 'Edad_Estructura_Anios', 'Precipitacion_Anual_mm', 'Humedad_Prom_Anual_pct', 'PM10_Promedio_ugm3', 'Distancia_Via_Principal_m', 'Acid_Rain_Index', 'Vibration_Risk' ] scaler = MinMaxScaler() df[numeric_cols] = scaler.fit_transform(df[numeric_cols]) # Result: all variables in range [0.0, 1.0]
